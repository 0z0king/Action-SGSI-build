# SGSI-build-action

## 使用方法:
```
1. fork此仓库
2. 编辑SGSI_Build.yml:
     ROM_URL: 为你的下载连接 
     ZIP_NAME: 为你下载的包名 
     OS_TYPE: 为你要跑的rom类型  
     BUILD_TYPE: 为你需要build的类型(ab|a-only) 
     REPACK_NAME: 为你需要打包的包名
3. 修改完毕后依次点击Start commit -> Actions选项卡 -> Star -> SGSI-Build
4. 上传好后的下载链接，留意输出结果 Download Link:
```
